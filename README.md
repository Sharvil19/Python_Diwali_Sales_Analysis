## 📌 Project Overview
This project analyzes Diwali sales data to understand customer purchasing behavior and identify key insights to improve business performance.

---

## 🛠️ Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Jupyter Notebook

---

## 📊 Key Analysis Performed
- Data Cleaning (handling null values, removing irrelevant columns)
- Exploratory Data Analysis (EDA)
- Customer segmentation (Age, Gender, Marital Status)
- State-wise and category-wise sales analysis

---

## 🔍 Key Insights
- Female customers contribute more to total sales
- Age group **26–35** has the highest purchasing power
- Top states: Maharashtra, Uttar Pradesh, Karnataka
- Most sold categories: Clothing, Food, Electronics

---

## 📈 Business Recommendations
- Target marketing campaigns towards **married women aged 26–35**
- Focus on high-performing states for better ROI
- Increase inventory for high-demand product categories

---

## 🎯 Conclusion
The analysis highlights key customer segments and product categories driving sales. Businesses can use these insights to optimize marketing strategies and improve revenue during festive seasons.
